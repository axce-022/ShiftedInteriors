# SHIFTED INTERIORS — Official Website

Premium single-page website for **SHIFTED INTERIORS**, designed as a digital extension of the Brand Partnership Portfolio media kit.

## Stack

- [Astro](https://astro.build) — static, lightweight, SEO-friendly
- Vanilla CSS — editorial design system aligned with the media kit
- No backend, database, or CMS

## Sections

1. Hero
2. About (+ Performance metrics)
3. Featured Work
4. Why Brands Partner With Us
5. Contact

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321)

## Production Build

```bash
npm run build
npm run preview
```

Static output is generated in `dist/`.

## Assets

Images and the partnership portfolio PDF are sourced from `SHIFTED_INTERIORS_Brand_Partnership_Portfolio_2026_v1.pptx`.

## Design System

- **Typography:** Montserrat
- **Palette:** White, Black, Charcoal, Purple `#6A5CFF`, Coral `#FF6A88`
- **Aesthetic:** Editorial luxury — minimal, confident, premium
